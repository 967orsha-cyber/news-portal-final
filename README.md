# Доска объявлений для фанатского сервера MMORPG

## Описание проекта
Этот проект реализует доску объявлений для игроков.  
Пользователи могут:
- Регистрироваться и подтверждать email
- Создавать объявления в категориях (Танки, Хилы, ДД и т.д.)
- Оставлять отклики на объявления
- Принимать или удалять отклики на свои объявления
- Получать уведомления на email

## Важно!
В этом же проекте присутствует старый модуль `news` (новостной портал),  
но основное задание — модуль `bulletin` (доска объявлений).  
Главная страница открывает **раздел объявлений**.

## Запуск проекта
- Создать виртуальное окружение
- Установить зависимости: `pip install -r requirements.txt`
- Применить миграции: `python manage.py migrate`
- Запустить сервер: `python manage.py runserver`

## Технологии
- Django
- Celery + Redis
- django-allauth
- django-ckeditor
